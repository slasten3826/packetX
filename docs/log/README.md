# Log

Сквозной лог нужен для переходных мыслей,
которые еще не до конца кристаллизовались в один слой.

Сюда ложатся:

- техлог
- легенда
- временные матрицы
- сравнения `X11` / `Wayland` / `X12`
- промежуточные гипотезы

Если документ со временем становится устойчивым,
его можно перенести в один из уровней:

- `chaos/`
- `table/`
- `crystall/`
- `manifest/`
