# Manifest

Слой `manifest` описывает финальное явление графики в мир.

Сюда должны ложиться документы про:

- composition
- fullscreen path
- direct scanout
- visibility graph
- final scene emission
- output realization

Композитор в логике `X12` должен быть здесь,
а не владеть всем стеком целиком.
